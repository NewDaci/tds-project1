- This script scrapes GitHub users in Paris with over 200 followers and their repositories using the GitHub API.
- One interesting fact: many developers have diverse repositories with multiple languages, but JavaScript still dominates.
- Recommendation: developers should maintain active profiles with frequent repository updates to improve visibility in search results.

## Usage

Run the Python script to scrape data and save it to `users.csv` and `repositories.csv`. Ensure you have a valid GitHub API token and install the `requests` library.
